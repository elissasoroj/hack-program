# hack-program
sketchprompt program


### How to use the sketchprompt tool:

```bash
sketchprompt -h
```
```bash
usage: sketchprompt [-h] [--subject] [--animal] [--plant] [--object] [--chordata] 
					[--plantae] [--scheme] [--color] [--time] [--clock]

optional arguments:
 -h, --help           show this help message and exit
 --prompt             returns full prompt
 --subject, -sub      returns random sketch subject
 --animal, -a         returns a random animal subject
 --plant, -p          returns a random plant subject
 --object, -ho        returns a random household object
 --chordata, --gbifa  returns a random animal species from GBIF
 --plantae, --gbifp   returns a random plant species from GBIF
 --color, -col        returns a hue
 --scheme             returns a color scheme
 --time               returns a random time limit
 -c, --clock          starts a coundown clock (enter time in seconds)


```
